# Predictive Analytics Using Historical Data

## Objective
Build a predictive analytics model using historical sales data.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Features
- Data preprocessing
- Data visualization
- Linear regression prediction
- Model accuracy evaluation
- Future sales forecasting

## Output
The project predicts future sales trends using historical monthly sales data.

## Results
- Mean Absolute Error: 3.09
- R2 Score: 0.998
